Apache curator

Apache curator 是zookeeper的基于java写的客户端框架， 它有以下几个功能:

  1. 链接管理 (重试机制)
  2. 支持异步
  3. 配置中心 （服务注册)
  4. Watchers, 监听配置的变化， (服务发现)
  5. typed models (轻松实现序列化 / 反序列化)
  6. 领导选举
  7. 分布式锁
  8. 分布式计算器
